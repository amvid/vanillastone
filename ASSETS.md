## Card art (AI-generated, original)

The card art in `web/public/art/` is **generated by an AI image tool from original
prompts** written for this project (see `web/public/art/README.md` and
`.notes/art-prompts.md`). Each prompt describes a generic fantasy archetype and is
kept visually distinct from any specific copyrighted character or logo.

- **Tool:** OpenAI ChatGPT image generation (DALL·E / GPT-4o image model).
- **Terms:** OpenAI [Terms of Use](https://openai.com/policies/terms-of-use). Under
  those terms OpenAI assigns the user all its right, title, and interest in
  generated output, and permits use (including commercial). This project's use —
  non-commercial, self-hostable, open-source — is within them. You remain
  responsible for not generating content that infringes a third party's IP.
- **Ownership / license:** purely AI-generated images may not be copyrightable, so
  we make **no ownership claim** over the art. It is provided "as is" for use within
  this project; the [MIT License](LICENSE) covers the **code only**.
- **Prompts:** original, no IP names — `.notes/art-prompts.md` (gitignored).

If any generated image is found to resemble an existing character or asset, replace
it and note the swap.
